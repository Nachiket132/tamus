# Temus

> Sit. Think. Then build.

Blueprint your ideas before you open any AI tool. Temus is a desktop app that forces you to slow down and plan before prompting — the thinking layer between inspiration and execution.

## Features
- 6-section blueprint canvas (Spark, Problem, Vision, Questions, Resources, First Move)
- Tag-based AI prompts — Business plan, Product research, Learning roadmap, Side project launch plan, and more
- Lock mechanic — commit your thinking before you build
- Copy to AI — one click structured prompt ready to paste into Claude or ChatGPT
- Command palette (Cmd+K)
- Works fully offline — no backend, no account needed

## Run locally
    git clone https://github.com/nachiketp/temus
    cd temus
    npm install
    npm run electron:dev

## Web version only
    npm run dev

## Build .dmg for Mac
    npm run electron:build

## Stack
React · Vite · Electron · localStorage
